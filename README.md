# giraffe1972

2D aircraft shooting game project.

## Files

- `index.html`: main game file
- `outputs/skyfront-1949.html`: previous generated HTML copy

## Run

Open `index.html` directly, or run a local web server:

```sh
npm start
```

Then open:

```text
http://localhost:5174/
```

For another device on the same Wi-Fi, use this computer's local IP address with port `5174`.
